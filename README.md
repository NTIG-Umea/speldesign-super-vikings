# super-vikings-spel

Phaser game

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build:prod
```

To create a development build:

```sh
npm run build:dev
```

## Running

```sh
open index.html in either root (~) or the dev folder (~/dist/dev)
```
